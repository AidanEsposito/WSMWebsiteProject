Welcome to the Github of the Wine Storage Management Page Recreation Project.

As a current employee of the company, I challenged myself with my cs degree and time to make a version of the site that is not only accurate to the current version, but has additional features and changes to improve its design.

Current site for reference: [https://www.winestoragemanagement.com/]

Future new site (if launched on a future date):

To start with this project, there are these webpages:

* Home: Hosts the main page of the site + links
* About: Gives details on the comapany and its practices
* Facility Features: Details the different aspects of each location
* Receiving Service: Details the wine moving program + lockers / rooms
* Storage + FAQ: Details how to store the wine + common questions
* Contact: Simple contact form for contacting management
* Member Portal: Portal base to pay rent / see locker info
* Blog: Blog about wine and WSM activities
* Friends of WSM: WSM partners + notible clients

This project is also planned to be expanded upon in the future now with a base layout including (not official) these ideas:

- Possible portal for locker information + facility info
- Updated "payment" site and template
- More advanced contact form that could connect to emails + texts
- Authentication and online blog services for customers 
- General website improvements + cleaning


To install the project for yourself, just follow these steps:

1. Install Git

This can be done at the Git official website linked here. Follow the guide tailored to your specific Operating System.

[https://git-scm.com/downloads]

2. Clone the repository

3. run the "git clone" command in the terminal to download

4. Install the following software:

- npm install 

- npm install react-router-dom

5. Use npm run dev to launch the program in a browser
